# Logo File Location

**IMPORTANT:** Place your `SchoolLogo.png` file in this directory.

The logo file should be named exactly: `SchoolLogo.png`

After adding the file:
1. Commit it to git: `git add public/images/SchoolLogo.png`
2. Push to GitHub: `git push origin main`
3. Vercel will automatically redeploy

The logo will be accessible at: `/images/SchoolLogo.png`
